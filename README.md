# ✍️ AI Blog Generator

An AI-powered Blog Generator built using **Python**, **Streamlit**, and **Groq LLMs** that creates high-quality blog posts with customizable length and writing style. The application generates complete articles including titles, introductions, main content, and conclusions based on user-provided topics.

---

## 🚀 Features

✅ Generate complete blog articles

✅ Multiple writing tones

✅ Customizable blog length

✅ AI-generated titles

✅ Automatic introductions and conclusions

✅ SEO-friendly content generation

✅ Interactive Streamlit interface

✅ Powered by Groq LLM

---

## 🛠️ Tech Stack

- Python 3.11
- Streamlit
- Groq API
- Python Dotenv
